# Поиск Исполнителей — Руководство пользователя

## Как открыть

Боковое меню → **Маркетинг** → **Поиск Исполнителей**

---

## 1. База монтажников

Вкладка содержит полный список всех монтажников (действующих и потенциальных).

### Добавление исполнителя

1. Нажать **+ Добавить**
2. Заполнить обязательные поля:
   - **Наименование** — ФИО или название организации
   - **ИНН** — идентификационный номер
3. Заполнить контакты: телефон, email, Telegram, WhatsApp
4. Выбрать **специализации** (вентиляция, кондиционирование и т.д.)
5. Указать город и регион
6. Указать расценки (ставка/час, ставка/день), размер бригады, стаж
7. Нажать **Создать**

### Фильтрация и поиск

- **Поиск** — по имени, телефону, email, городу
- **Фильтры** (кнопка «Фильтры»):
  - Город
  - Статус: потенциальные / действующие
  - Доступность: доступные / недоступные

### Просмотр деталей

Кликните по строке в таблице. Откроется панель с:
- Полной контактной информацией
- Специализациями
- Расценками и опытом
- **Историей контактов** — все звонки, письма, встречи

### Добавление записи контакта

1. В деталях исполнителя нажать **Записать контакт**
2. Выбрать канал (телефон, email, встреча и т.д.)
3. Указать направление (входящий/исходящий)
4. Написать тему и комментарий
5. Нажать **Сохранить**

---

## 2. Авито

### Входящие объявления

Здесь хранятся объявления монтажников, найденные на Avito.

**Добавление объявления вручную:**
1. Нажать **+ Добавить объявление**
2. Вставить ссылку на Avito (или заполнить данные вручную)
3. Указать заголовок, город, имя продавца
4. Нажать **Добавить**

**Действия с объявлениями:**
- 👁 — отметить как просмотренное
- 👤+ — конвертировать в исполнителя (создаёт запись в базе монтажников)
- ✕ — отклонить (не подходит)

### Наши объявления

Список МП, опубликованных как объявления на Avito. Показывает:
- Статус публикации
- Просмотры, контакты, избранное
- Ссылку на объявление

---

## 3. Рассылки

### Создание рассылки

1. Нажать **+ Создать рассылку**
2. Заполнить:
   - **Название** — для идентификации
   - **Тип** — Email или SMS
   - **Тема** (только для Email)
   - **Текст сообщения**
3. Настроить **фильтры получателей**:
   - Специализации
   - Города
   - Статус (потенциальные / действующие)
   - Доступность
4. Нажать **Создать черновик**

### Предпросмотр

Перед отправкой нажмите 👁 на рассылке — увидите:
- Количество получателей
- Список первых 20 получателей
- Примерную стоимость SMS

### Отправка

1. Нажать ✈ (отправить)
2. Подтвердить отправку
3. Рассылка перейдёт в статус «Отправляется» → «Завершено»

---

## 4. История контактов

Глобальная лента всех коммуникаций — звонки, письма, SMS, встречи. Записи создаются:
- Автоматически при отправке рассылок
- Вручную через профиль исполнителя

---

## 5. Настройки

### Avito API

1. Получите Client ID и Client Secret на https://developers.avito.ru/
2. Введите их в поля настроек
3. Включите флаг «Интеграция активна»
4. Опционально: включите «Авто-публикация МП» — МП будут публиковаться на Avito при смене статуса на «Опубликовано»

### Шаблон объявления

Настройте текст объявления с переменными:
- `{object_name}` — название объекта
- `{city}` — город объекта
- `{work_types}` — виды работ
- `{man_hours}` — человеко-часы
- `{total_amount}` — сумма

### Ключевые слова

Список слов для мониторинга Avito (вентиляция, кондиционирование и т.д.). Добавляйте и удаляйте по необходимости.

### Unisender

1. Зарегистрируйтесь на https://www.unisender.com/
2. Получите API-ключ в настройках
3. Введите API-ключ, email и имя отправителя
4. Для SMS укажите имя отправителя (до 11 символов)
5. Включите флаг «Активен»
