# Руководство по настройке критериев рейтинга

## Как работает рейтинг

Система AI-рейтинга автоматически оценивает черновики новостей по шкале 0-5 звёзд.
Критерии хранятся в базе данных и редактируются через веб-интерфейс (ERP → HVAC → Критерии рейтинга).

## Уровни звёзд

### 5★ — Интересно
Новости, которые безусловно публикуются. Автоматически:
- Любые новости из российских источников или о российском рынке

### 4★ — Ограниченно интересно
Новости, которые оператор должен просмотреть. Автоматически:
- Любые новости от КМП (крупных мировых производителей)
- Новости, найденные в 2+ источниках (дубликаты)
- Новые продукты, технологии, R&D
- Новости от правительства/администрации

### 3★ — Не интересно
Пропускаются при быстром просмотре:
- Назначения/увольнения персон
- Партнёрства/слияния (не КМП)
- Выставочные стенды
- Зелёный переход, декарбонизация
- Конкурсы, испытания, бюллетени
- Форумы, курсы, вебинары
- И т.д.

### 2★ — Не по теме
Новости, не имеющие отношения к HVAC:
- Канализация, водоснабжение, отопление, котлы
- Случайно попавшие не по теме

### 0★ — Не классифицировано
Новости, не подходящие ни под один критерий. Требуют ручного разбора.

## Как добавить новый критерий

1. Откройте ERP → HVAC → Критерии рейтинга
2. Найдите нужный уровень звёзд
3. Нажмите "Добавить"
4. Заполните:
   - **Название** — короткое имя для списка
   - **Описание** — подробное описание для LLM. Чем точнее, тем лучше работает классификация
   - **Ключевые слова** — подсказки (через запятую)
5. Сохраните

## Двухуровневые критерии

Пример: "Партнёрства компаний" → 3★, НО "Партнёрства КМП" → 4★

Для этого:
1. Создайте родительский критерий "Партнёрства компаний" на уровне 3★
2. Создайте дочерний критерий "Партнёрства КМП" с `override_star_rating = 4`

Если LLM определит, что новость подходит под оба — применится override рейтинг (4★).

## КМП — Крупные мировые производители

Список КМП управляется через страницу Производителей (флаг "KMP").
Любые новости от КМП автоматически получают минимум 4★ (до запуска LLM).

Начальный список заполняется командой: `python manage.py populate_kmp`

## Пересчёт рейтинга

После изменения критериев можно перезапустить рейтинг:
- ERP → HVAC → Настройки рейтинга → "Оценить все неоценённые"
- Или сбросить рейтинг через Django admin и запустить заново
