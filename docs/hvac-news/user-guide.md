# HVAC-новости: руководство пользователя

## Навигация

Раздел доступен в меню: **HVAC-новости > Новости** (`/hvac/news`).

## Просмотр новостей

Страница отображает все новости в виде карточек с превью изображения, заголовком, датой и кратким текстом.

### Фильтрация по статусу (для администраторов)

Выпадающий список в правом верхнем углу:
- **Все новости** — без фильтра
- **Опубликованные** — видны всем пользователям
- **Черновики** — только для админов, результат автоматического поиска
- **Запланированные** — отложенная публикация

## Массовое удаление

Для быстрой работы с черновиками доступно массовое удаление:

1. **Выбор новостей** — чекбокс слева от каждой карточки
2. **Выбрать все** — чекбокс в шапке списка выделяет все новости на странице
3. **Кнопка "Удалить выбранные (N)"** — появляется при выделении, показывает счётчик
4. **Подтверждение** — диалог перед удалением, действие необратимо

Выделенные карточки подсвечиваются рамкой.

## Действия с отдельной новостью

Для каждой новости (в нижней панели карточки):
- **Редактировать** — открывает редактор
- **Удалить** — удаление с подтверждением
- **Источник** — ссылка на оригинальную статью (для автоматически найденных)

## Создание новости

Кнопка **"Создать новость"** в правом верхнем углу.
