# Первый рефакторинг

## Выполненные улучшения

### 1. Создана базовая абстрактная модель
- **Файл**: `core/models.py`
- **Класс**: `TimestampedModel`
- **Цель**: Устранение дублирования полей `created_at` и `updated_at` в моделях
- **Результат**: Модели `Object` и `Contract` теперь наследуются от `TimestampedModel`

### 2. Рефакторинг моделей
- **Object**: Удалены дублирующиеся поля `created_at` и `updated_at`, убран неиспользуемый импорт `User`
- **Contract**: Удалены дублирующиеся поля, убран комментарий `# Create your models here.`

### 3. Добавлена админка для Object
- **Файл**: `objects/admin.py`
- **Функциональность**: Список объектов с фильтрацией и поиском, отображение временных меток

### 4. Улучшены тесты
- **Object**: Добавлены 4 unit-теста (создание, уникальность, строковое представление, временные метки)
- **Результат**: Всего 7 тестов проходят успешно (3 для Contract + 4 для Object)

### 5. Структурные улучшения
- Создано приложение `core` для общих компонентов
- Добавлено в `INSTALLED_APPS`
- Улучшена организация кода

## Метрики
- **Тесты**: 7 тестов (было 3, добавлено 4)
- **Модели**: 2 модели используют базовый класс
- **Админка**: 2 модели зарегистрированы в админке
- **Дублирование кода**: Устранено в полях временных меток
