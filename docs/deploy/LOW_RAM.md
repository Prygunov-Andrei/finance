# Low-RAM deployment (2GB)

Цели:
- избежать OOM во время сборки образов и миграций;
- минимизировать риск падений из-за отсутствия swap.

## Рекомендованный сценарий

На сервере:

```bash
cd /opt/finans_assistant
./deploy/deploy_lowram_safe.sh
```

## Практические советы

- Swap: если swap < 1GB, создать 2GB (скрипты деплоя умеют это делать).
- Сборка образов: избегать параллельной пересборки всего сразу; если нужно — собирать по одному сервису.
- Docker cache: следить за диском и аккуратно чистить, если заканчивается место.

## Если “зависает” на сборке

```bash
docker stats
free -h
df -h
docker compose -f docker-compose.prod.yml logs -f --tail=200
```
