# Пример диалога сметчика с LLM-агентом

Это репрезентативный сценарий: сметчик просит подобрать аналог для позиции сметы.

## Контекст

- Смета: «Вентиляция корпус А» (версия 1).
- Позиция, вокруг которой вопрос: «Вентилятор крышный MOB2600/45-3a», 4 шт., item_id=`9b810000-0000-4000-8000-000000000001`.
- Текущий подбор: ProductKnowledge (verified), confidence 0.87, work_item_id=wi-504 («Монтаж крышного вентилятора до 200 кг»).

## Транскрипт

**Сметчик:** Найди мне аналог этому вентилятору, у поставщика нет на складе.

**Агент (внутри: вызывает `get_item` с item_id=`9b81...`):**
```json
{
  "tool": "get_item",
  "arguments": {"item_id": "9b810000-0000-4000-8000-000000000001"}
}
```

**Система отдаёт агенту:**
```json
{
  "item_id": "9b81...",
  "name": "Вентилятор крышный MOB2600/45-3a",
  "model_name": "MOB2600/45-3a",
  "brand": "MOB",
  "unit": "шт",
  "quantity": 4,
  "tech_specs": {"flow": "2600 м³/ч", "power": "0.45 кВт", "voltage": "3x400V"},
  "current_match": {
    "product_id": "prod-1",
    "product_name": "Вентилятор крышный MOB2600/45-3a",
    "work_item_id": "wi-504",
    "source": "knowledge",
    "confidence": 0.87
  }
}
```

**Агент (рассуждение):** «Позиция — крышный вентилятор с расходом 2600 м³/ч, мощностью 0.45 кВт. Запрошу аналоги по этим параметрам.»

**Агент (внутри: вызывает `find_alternatives_by_specs`):**
```json
{
  "tool": "find_alternatives_by_specs",
  "arguments": {"item_id": "9b810000-0000-4000-8000-000000000001", "count": 5}
}
```

**Система отдаёт агенту:**
```json
{
  "query_specs": {"flow": "2600 м³/ч", "power": "0.45 кВт"},
  "alternatives": [
    {
      "product_id": "prod-42",
      "product_name": "Вентилятор крышный Systemair MOBS-063",
      "brand": "Systemair",
      "latest_price": 78500.00,
      "price_source": "supplier_catalog",
      "tech_specs": {"flow": "2700 м³/ч", "power": "0.5 кВт"},
      "match_score": 0.92,
      "reasoning": "Расход в пределах 4%, мощность выше на 11%"
    },
    {
      "product_id": "prod-89",
      "product_name": "Вентилятор крышный Вентс ВКОК 2.7",
      "brand": "VENTS",
      "latest_price": 65000.00,
      "price_source": "price_history",
      "tech_specs": {"flow": "2800 м³/ч", "power": "0.55 кВт"},
      "match_score": 0.88,
      "reasoning": "Расход выше на 7%, мощность выше на 22%; ниже цена"
    },
    {
      "product_id": "prod-134",
      "product_name": "Вентилятор крышный Арктос ВКР-063",
      "brand": "Арктос",
      "latest_price": 72000.00,
      "price_source": "supplier_catalog",
      "tech_specs": {"flow": "2500 м³/ч", "power": "0.4 кВт"},
      "match_score": 0.87,
      "reasoning": "Расход ниже на 4%, мощность ниже на 11%"
    }
  ],
  "total_found": 8,
  "returned": 3
}
```

**Агент (ответ сметчику):**

Нашёл три близких аналога:

1. **Systemair MOBS-063** — 78 500 ₽, расход 2700 м³/ч (+4%), мощность 0.5 кВт. Максимально близкий по характеристикам, дороже прочих.
2. **Вентс ВКОК 2.7** — 65 000 ₽, расход 2800 м³/ч (+7%), мощность 0.55 кВт. Самый недорогой, но с запасом по производительности и потребляемой мощности.
3. **Арктос ВКР-063** — 72 000 ₽, расход 2500 м³/ч (–4%), мощность 0.4 кВт. Чуть меньше производительность; стоит, если допустим минимальный недобор.

Если нужна полная замена без проверки совместимости по шумовым характеристикам и габаритам — возьми Systemair MOBS-063. Если важна цена и запас мощности не критичен — Арктос ВКР-063. Если экономия важнее запаса по мощности — Вентс ВКОК 2.7.

Чтобы применить замену, нажми «Заменить» рядом с нужной опцией в панели предложений. Я не меняю смету сам.
