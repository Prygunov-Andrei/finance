# HVAC-новости — Руководство оператора

## Общий обзор

Раздел HVAC-новости обеспечивает автоматический мониторинг мировых новостей в области вентиляции, кондиционирования и холодоснабжения. Система находит новости из сотен источников, оценивает их важность с помощью AI, обнаруживает дубликаты и публикует на портале hvac-info.com.

---

## Новости

### Просмотр и фильтрация

На странице **Новости** отображаются все найденные новости. Доступные фильтры:

- **По статусу** — Все / Опубликованные / Черновики / Запланированные
- **По рейтингу** — кнопки 0-5 звёзд (множественный выбор)

При просмотре черновиков новости автоматически группируются по рейтингу — от 5 звёзд (самые интересные) до 0 (не классифицированные). Каждая группа сворачивается.

### Создание новости вручную

1. Нажмите **"Создать новость"**
2. Заполните заголовок и текст (поддерживается форматирование, вставка изображений)
3. Выберите исходный язык
4. Установите дату публикации и статус
5. При необходимости включите **"Автоперевод"** — система переведёт на все языки

### Редактирование и публикация

- Нажмите на новость для просмотра деталей
- Кнопка **"Редактировать"** открывает редактор
- Кнопка **"Опубликовать"** немедленно публикует новость на портале
- Рейтинг можно изменить вручную — кликните на badge со звёздами

### Массовые операции

- Отметьте новости чекбоксами
- Используйте **"Удалить выбранные"** для массового удаления
- В группированном виде черновиков: **"Выбрать все"** выделяет всю группу, **"Удалить группу"** — удаляет все 1-звёздные

## AI-рейтинг новостей

### Шкала оценки

| Звёзды | Значение | Действие оператора |
|--------|----------|-------------------|
| 5★ | Интересно | Публиковать |
| 4★ | Ограниченно интересно | Просмотреть, решить |
| 3★ | Не интересно | Пропустить |
| 2★ | Не по теме | Удалить |
| 1★ | Новостей не найдено | Удалить (служебная запись) |
| 0★ | Не классифицировано | Разобрать вручную, дополнить критерии |

### Автоматические правила (без LLM)

Некоторые новости оцениваются мгновенно, без обращения к AI:

- **Российские источники** (source_language=ru) → автоматически **5★**
- **Новости от КМП** (крупные мировые производители) → минимум **4★**
- **"Новостей не найдено"** (служебные записи) → автоматически **1★**

### Запуск рейтинга

1. Откройте **Настройки рейтинга**
2. Нажмите **"Оценить все неоценённые"**
3. Наблюдайте за прогрессом — фаза, количество обработанных, стоимость
4. По завершении появится распределение по звёздам

Рейтинг также запускается автоматически после каждого поиска новостей.

## Настройки рейтинга

### Конфигурация провайдера

Каждая конфигурация определяет, какой LLM-провайдер и модель используются для оценки.

**Создание конфигурации:**
1. Нажмите **"Создать конфигурацию"**
2. **Шаг 1 — Провайдер**: выберите основной (Grok, Claude, Gemini, OpenAI) и резервные
3. **Шаг 2 — Модель**: выберите между экономным и мощным вариантом. Цены заполнятся автоматически
4. **Шаг 3 — Промпты**: при необходимости отредактируйте инструкции для AI
5. **Шаг 4 — Тарифы**: проверьте цены за токены
6. **Шаг 5 — Проверка**: убедитесь что провайдер доступен, сохраните

**Переключение между конфигурациями:** нажмите галочку (Activate) на нужной строке.

### Параметры

- **Температура** (0.0-1.0) — ниже = точнее и стабильнее, выше = креативнее. Для рейтинга рекомендуется 0.1-0.3
- **Размер батча** — сколько новостей отправлять в одном запросе к AI. Больше = дешевле, но менее точно
- **Порог дубликатов** (0.5-0.95) — минимальная схожесть заголовков для объединения. Выше = строже

## Критерии рейтинга

### Структура

Критерии организованы по уровням звёзд. AI использует их как инструкцию — на что обращать внимание при оценке.

Каждый критерий содержит:
- **Имя** — короткое название для списка
- **Описание** — подробная инструкция для AI (чем точнее, тем лучше)
- **Ключевые слова** — подсказки для поиска
- **Активность** — выключенные критерии не участвуют в оценке

### Двухуровневые критерии

Некоторые критерии имеют условные уточнения. Например:

- Критерий "Партнёрства компаний" → 3★
- Дочерний критерий "Но если это КМП" → override на 4★

Если сработал и родительский, и дочерний — применяется рейтинг дочернего.

### Редактирование

1. Откройте **Критерии рейтинга**
2. Разверните нужный уровень звёзд
3. Нажмите **"+"** для добавления нового критерия
4. Используйте переключатель для активации/деактивации
5. Нажмите на значок редактирования для изменения
6. После изменения критериев **перезапустите рейтинг** для переоценки

## Поиск новостей (Discovery)

### Как работает

1. Система берёт список источников (сайтов) и производителей
2. Для каждого отправляет запрос к LLM с веб-поиском
3. LLM находит свежие новости, переводит на русский, извлекает данные
4. Создаются черновики новостей
5. Автоматически запускается AI-рейтинг

### Запуск поиска

**По источникам:**
- Откройте **Ресурсы** → выберите источники → **"Найти новости"**
- Или на странице отдельного ресурса нажмите **"Найти новости"**

**По производителям:**
- Откройте **Производители** → **"Найти новости по производителям"**

### Настройки поиска

- **Провайдер** — какой AI использовать для поиска (может отличаться от рейтинга)
- **Модель** — конкретная модель (Grok 4.1 Fast рекомендуется для поиска — у него встроенный веб-поиск)
- **Температура** — для поиска рекомендуется 0.2-0.4
- **Максимум новостей** — лимит на источник за один запуск

## Производители

### КМП (Крупные мировые производители)

Производители с флагом **КМП** получают особый статус:
- Любые новости от КМП автоматически получают минимум **4★**
- Это такие компании как DAIKIN, Mitsubishi, LG, Samsung, TRANE, Carrier и др.

**Управление списком:**
- На странице производителей видно все компании, сгруппированные по регионам
- Флаг КМП устанавливается через Django Admin или командой `python manage.py populate_kmp`

### Добавление производителя

1. Нажмите **"Добавить производителя"**
2. Укажите название, описание, регион
3. Добавьте до 3 сайтов — они используются для поиска новостей

## Публичный портал

### Фильтры для посетителей

На публичной странице hvac-info.com посетители видят:
- По умолчанию — только новости с рейтингом **5★**
- Могут включить показ 4★, 3★, 2★ через фильтр звёзд
- Фильтр по месяцу/году
- Фильтр по региону

### Отображение

- Новости с 4★ и выше показывают badge со звёздами
- Новости с указанным производителем показывают badge производителя
- Доступны режимы просмотра: список и плитка

## Аналитика

### Статистика рейтинга

В разделе **Настройки рейтинга** видна сводная статистика:
- Количество запусков рейтинга
- Общее число оценённых новостей
- Суммарная стоимость
- Последние запуски с распределением по звёздам

### Анализ опубликованных

Кнопка **"Анализ опубликованных"** отправляет уже опубликованные новости в AI для выявления паттернов. Результаты помогут уточнить критерии — какие темы вы считаете интересными, но забыли добавить в критерии.

## Частые вопросы

### Почему много новостей с 0★?

Значит AI не смог соотнести их ни с одним критерием. Нужно:
1. Просмотреть эти новости вручную
2. Определить, какие критерии не хватает
3. Добавить новые критерии в редакторе
4. Перезапустить рейтинг

### Как сменить AI-провайдер?

1. Откройте **Настройки рейтинга**
2. Нажмите **"Создать конфигурацию"** или отредактируйте существующую
3. Выберите другой провайдер и модель
4. Активируйте новую конфигурацию
5. Запустите рейтинг заново

### Как работают дубликаты?

Система сравнивает заголовки новостей. Если схожесть выше порога (по умолчанию 75%), новости объединяются в группу и получают минимум 4★ — раз новость повторяется в нескольких источниках, она важна.

### Можно ли изменить рейтинг вручную?

Да. В списке новостей кликните на badge со звёздами — появится панель выбора рейтинга от 0 до 5.
