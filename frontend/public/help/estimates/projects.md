# Проекты ПТО

## Создание проекта

1. Перейдите в **ПТО > Проекты**
2. Нажмите **Создать проект**
3. Заполните обязательные поля: шифр, дата, название, стадия, объект
4. При необходимости прикрепите файлы и укажите их тип

Если нужного объекта нет — нажмите **+** рядом с подписью «Объект» для быстрого создания.

## Замечания

На вкладке **Замечания** хранится журнал замечаний к проекту.

### Форматирование

Замечания поддерживают Markdown:

- `**жирный**` — **жирный**
- `*курсив*` — *курсив*
- `# Заголовок` — заголовок
- `- пункт` — маркированный список
- `` `код` `` — inline-код
- Таблицы в формате GFM

При создании или редактировании замечания переключайтесь между вкладками **Редактор** и **Предпросмотр**.

### Экспорт

Нажмите **Скачать .md** для выгрузки всех замечаний одним файлом.

## Файлы проекта

К проекту можно прикреплять файлы разных типов (Спецификация, Графическая часть и др.). Файлы проектов доступны для импорта строк в связанные сметы.
